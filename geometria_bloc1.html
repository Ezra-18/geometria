Tema 1 · Vectors

Vectors en l'Espai:
Conceptes Base

El fonament de tota la geometria analítica.

📐

Definició

Un vector és un segment orientat amb mòdul (longitud), direcció i sentit. Es defineix donant un origen A i un extrem B.

Vector AB AB⃗ = B − A = (x₂−x₁, y₂−y₁, z₂−z₁)

Regla d'or: sempre extrem menys origen. Mai al revés.

🔧

Operacions bàsiques

Suma i resta u⃗ + v⃗ = (x₁+x₂, y₁+y₂, z₁+z₂)
u⃗ − v⃗ = (x₁−x₂, y₁−y₂, z₁−z₂)
Producte per escalar k k · u⃗ = (k·x₁, k·y₁, k·z₁) → mateixa direcció, sentit igual (k>0) o oposat (k<0)
🎯

Vectors proporcionals (paral·lels)

Dos vectors u⃗ i v⃗ són proporcionals si existeix un k tal que u⃗ = k·v⃗. Això significa que les seves components guarden la mateixa relació:

u⃗ ∥ v⃗ ↔ x₁/x₂ = y₁/y₂ = z₁/z₂ (= k)
Dos vectors paral·lels apunten en la mateixa direcció. Aquesta propietat és CLAU per determinar posicions relatives de rectes i plans.
Tema 2 · Mòdul

Mòdul i
Vector Unitari

Mesurar la "longitud" d'un vector.

📏

Fórmules clau

Mòdul |u⃗| = √(x²+y²+z²)
Vector unitari (mateixa direcció, mòdul = 1) û = u⃗ / |u⃗| = (x/|u⃗|, y/|u⃗|, z/|u⃗|)
⚙ Algoritme: calcular el vector unitari
1
Identificar el vector Escriu les components: u⃗ = (x, y, z)
2
Calcular el mòdul |u⃗| = √(x²+y²+z²). Simplifica l'arrel si pots.
3
Dividir cada component û = (x/|u⃗|, y/|u⃗|, z/|u⃗|)
4
Verificació Comprova que |û| = 1. Si no dona 1, hi ha un error.
Tema 3 · ★ Alta prioritat PAU

Producte Escalar
i Angle entre Vectors

La porta d'entrada a angles, perpendicularitat i molt més.

✖️

Definició i càlcul

El producte escalar de dos vectors dona un nombre real (no un vector).

Fórmula geomètrica u⃗ · v⃗ = |u⃗| · |v⃗| · cos(α)
Fórmula analítica (base ortonormal) ← la que uses a PAU u⃗ · v⃗ = x₁x₂ + y₁y₂ + z₁z₂
Angle entre dos vectors cos(α) = (u⃗ · v⃗) / (|u⃗| · |v⃗|)
⚡

Casos especials importants

CondicióResultatInterpretació
u⃗ · v⃗ = 0α = 90°Vectors perpendiculars
u⃗ · v⃗ > 0α agutAngle menor de 90°
u⃗ · v⃗ < 0α obtúsAngle major de 90°
u⃗ · u⃗= |u⃗|²Producte escalar d'un vector per ell mateix
La perpendicularitat (producte escalar = 0) és l'eina que s'usa milers de vegades: per trobar normals a plans, per calcular distàncies, per verificar angles rectes.
⚙ Algoritme: angle entre dos vectors
1
Calcular el producte escalar u⃗ · v⃗ = x₁x₂ + y₁y₂ + z₁z₂
2
Calcular els dos mòduls |u⃗| = √(x₁²+y₁²+z₁²) i |v⃗| = √(x₂²+y₂²+z₂²)
3
Aplicar la fórmula cos(α) = (u⃗·v⃗) / (|u⃗|·|v⃗|) → α = arccos(resultat)
4
Interpretar el resultat Si cos = 0 → perpendiculars. Si |cos| = 1 → paral·lels.
★ Exemple PAU

Angle entre vectors i perpendicularitat

Donats u⃗ = (2, −1, 3) i v⃗ = (1, k, −1).
a) Calcula k perquè u⃗ i v⃗ siguin perpendiculars.
b) Si k = 0, calcula l'angle que formen.
a)
Perpendiculars ↔ u⃗·v⃗ = 0
u⃗·v⃗ = 2·1 + (−1)·k + 3·(−1) = 2 − k − 3 = −1 − k = 0
→ k = −1
b)
k = 0 → v⃗ = (1, 0, −1)
u⃗·v⃗ = 2·1 + (−1)·0 + 3·(−1) = 2 − 3 = −1
|u⃗| = √(4+1+9) = √14    |v⃗| = √(1+0+1) = √2
cos(α) = −1 / (√14·√2) = −1/√28 ≈ −0.189
→ α = arccos(−0.189) ≈ 100.9°
Tema 4 · Projecció

Projecció d'un
Vector sobre un altre

La "ombra" d'un vector sobre la direcció d'un altre.

🔦

Fórmules

Donat u⃗ = (x₁,y₁,z₁) sobre v⃗ = (x₂,y₂,z₂):

Segment projecció (escalar) projv⃗(u⃗) = (u⃗ · v⃗) / |v⃗|
Vector projecció proj⃗v⃗(u⃗) = [(u⃗·v⃗) / |v⃗|²] · v⃗
El vector projecció és la "component" de u⃗ en la direcció de v⃗. S'usa en distàncies i per descompondre forces en física.
⚙ Algoritme: vector projecció
1
Producte escalar → u⃗ · v⃗
2
Mòdul al quadrat de v⃗ → |v⃗|² = x₂²+y₂²+z₂²
3
Multiplicar per v⃗ → [(u⃗·v⃗)/|v⃗|²] · (x₂,y₂,z₂)
Tema 5 · ★ Alta prioritat PAU

Producte Vectorial
i Àrees

Genera un vector perpendicular als dos. Imprescindible per a plans i àrees.

✖

Definició i càlcul

El producte vectorial u⃗ × v⃗ dona un nou vector, perpendicular als dos:

Determinant 3×3 (mnemotècnica) u⃗×v⃗ = | î ĵ k̂ |
          | x₁ y₁ z₁ |
          | x₂ y₂ z₂ |

= (y₁z₂−z₁y₂, z₁x₂−x₁z₂, x₁y₂−y₁x₂)
Mòdul = àrea del paral·lelogram |u⃗×v⃗| = Àrea paral·lelogram
Àrea triangle = ½ · |u⃗×v⃗|
🎯

Usos a PAU

El necessites per...Com s'usa
Vector normal a un plaAgafes 2 vectors directors del pla → u⃗×v⃗ és el normal
Àrea d'un triangle½·|AB⃗×AC⃗|
Àrea d'un paral·lelogram|u⃗×v⃗|
Distància punt-recta|RP⃗×d⃗| / |d⃗|
⚙ Algoritme: producte vectorial per determinants
1
Escriu el determinant 3×3 Fila 1: î ĵ k̂ | Fila 2: x₁ y₁ z₁ | Fila 3: x₂ y₂ z₂
2
Component î (cofactor de la 1a columna) +|y₁ z₁; y₂ z₂| = y₁z₂ − z₁y₂
3
Component ĵ (signe negatiu!) −|x₁ z₁; x₂ z₂| = −(x₁z₂ − z₁x₂) = z₁x₂ − x₁z₂
4
Component k̂ +|x₁ y₁; x₂ y₂| = x₁y₂ − y₁x₂
5
Verificació bàsica Comprova que (u⃗×v⃗)·u⃗ = 0 i (u⃗×v⃗)·v⃗ = 0 (ha de ser perpendicular als dos)
La component ĵ porta signe NEGATIU perquè el determinant s'expandeix per la primera fila amb signes +−+. Molts errors a PAU venen d'oblidar aquest signe.
★ Exemple PAU

Àrea d'un triangle donats tres vèrtexs

Donats els punts A(1, 0, 2), B(3, 1, −1), C(0, 2, 1).
Calcula l'àrea del triangle ABC.
1
Vectors dels costats des de A:
AB⃗ = B−A = (2, 1, −3)
AC⃗ = C−A = (−1, 2, −1)
2
Producte vectorial AB⃗ × AC⃗:
î: (1·(−1) − (−3)·2) = −1+6 = 5
ĵ: −(2·(−1) − (−3)·(−1)) = −(−2−3) = 5
k̂: (2·2 − 1·(−1)) = 4+1 = 5
AB⃗×AC⃗ = (5, 5, 5)
3
|AB⃗×AC⃗| = √(25+25+25) = √75 = 5√3
Àrea = ½ · 5√3 = 5√3/2 ≈ 4.33 u²
Tema 6 · ★ Alta prioritat PAU

Producte Mixt
i Volums

Tres vectors, un determinant, moltes aplicacions.

📦

Definició i càlcul

El producte mixt de u⃗, v⃗, w⃗ és el determinant 3×3 format per les seves components:

Producte mixt = determinant [u⃗, v⃗, w⃗] = | x₁ y₁ z₁ |
               | x₂ y₂ z₂ |
               | x₃ y₃ z₃ |
Volum del paral·lelepípede Vparal·lelepípede = |[u⃗, v⃗, w⃗]|
Volum del tetraedre (4 vèrtexs) Vtetraedre = (1/6) · |[AB⃗, AC⃗, AD⃗]|
💎

Aplicació CRÍTICA: vectors coplanaris

Si el producte mixt = 0, els tres vectors (o punts) són coplanaris. Aquesta propietat serveix per:

Producte mixtSignificat
[u⃗,v⃗,w⃗] = 0Vectors coplanaris (estan al mateix pla)
[u⃗,v⃗,w⃗] ≠ 0NO coplanaris (formen un volum real)
Rectes r, s: det = 0Les rectes es tallen (o coincideixen)
Rectes r, s: det ≠ 0Les rectes es creuen a l'espai (no es tallen)
Quan estudiem posicions relatives de rectes al Bloc 2, el producte mixt és l'eina definitiva per distingir entre "es tallen" i "es creuen". Memoritza: det=0 → coplanaris → es tallen.
⚙ Algoritme: volum d'un tetraedre ABCD
1
Tria un vèrtex base (A) Calcula: AB⃗, AC⃗, AD⃗
2
Munta el determinant 3×3 Files = components de AB⃗, AC⃗, AD⃗
3
Calcula el determinant Usa la regla de Sarrus o expansió per la primera fila
4
Multiplica per 1/6 i pren el valor absolut V = (1/6) · |det|
★ Exemple PAU

Volum del tetraedre i coplanaritat

Donats A(1,1,1), B(2,3,0), C(0,1,2), D(3,2,1).
a) Calcula el volum del tetraedre ABCD.
b) Els 4 punts, ¿formen un tetraedre o són coplanaris?
1
Vectors des de A:
AB⃗ = (1, 2, −1)
AC⃗ = (−1, 0, 1)
AD⃗ = (2, 1, 0)
2
Determinant 3×3:
| 1 2 −1 |
|−1 0 1 |
| 2 1 0 |

= 1(0·0−1·1) − 2((−1)·0−1·2) + (−1)((−1)·1−0·2)
= 1(−1) − 2(−2) + (−1)(−1)
= −1 + 4 + 1 = 4
3
V = (1/6)·|4| = 4/6 = 2/3 u³
Com det ≠ 0 → els 4 punts NO són coplanaris → formen un tetraedre real.
